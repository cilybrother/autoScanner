autoScanner
===========

GUI of Twain and SANE API with auto scanning.

Requirements:
------------
* Python
* PyGI (PyGObject Introspection)
 * GTK
* pyScanLib
 * SANE (Linux)
 * TWAIN (Win32)


Features:
----------
* Auto Scanning
* Selected Area Scanning
* Convert Scanned Image to Text (OCR)
* Multi Platform (Windows, Linux)
